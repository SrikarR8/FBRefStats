## Contents

1. **FbRef**

   - Includes yearly data by player for the following:
  
     - Summary Stats (Goals, Assists, xG)
     - Playing Time
     - Possession
     - Standard Goalkeeping
     - Advanced Goalkeeping (post-shot XG saved) 
     - Defense 
     - Disipline and Aerial
     - Creation (Goal and Shot creating actions)
     - Passing
     - Pass Types
     - Shooting
2. **Transfermarkt**

   - Transfer Data 
   - Market Valuations (Present and Historic)
